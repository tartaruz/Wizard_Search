### Installation

_Wizard Search_ requires [Python](https://nodejs.org/) 3+ to run.

Install the dependencies ...

```sh
$ pip3 install -r requirements.txt 
```

Set up the Flask application and run it..

```sh
$ export FLASK_APP=hello.py
$ flask run
```
