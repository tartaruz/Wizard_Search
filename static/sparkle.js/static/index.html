sparkle.js
An animation based sparkling idea

Welcome to sparkle.js

Getting started

Load sparkle.min.js and add it to your project. Locate you HTMLElement via JS ans .sparkle() it!

To get the beautiful Header Sparkling effect, try this: 

    document.getElementById("sparkle").sparkle({
        position: "afterbegin",
        stars: 25,
        minAge: 500,
        maxAge: 2000,
        size: 10
    });

Documentation

Before we begin, please note, that sparkle.js enforces position: relative if the requested element does not have fixed, absolute or relative set.

You can select your elements with your well known JS Syntax. Important is, that your selector returns an HTMLElement or NodeList.

Select an element with Children:

    document.querySelector("#example-1").sparkle();

Or select multiple Elements at once:

    document.querySelectorAll("#example-2>div").sparkle();

You can even use your own Strings or HTML

    var i = 0;
    document.querySelectorAll("#example-3>div").forEach(el => {
        i++
        el.sparkle({
            starsource: `${i}`,
            count: i
        });
    });

...and existing HTMLElements 

    document.querySelectorAll("#example-4").sparkle({
        starsource: document.querySelector(".smile"),
        count: 16,
        minAge: 300,
        maxAge: 700
    });
